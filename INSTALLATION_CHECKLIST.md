### הרצה
- [ ] לחצתי על "+ New Query"
- [ ] פתחתי את הקובץ: `supabase/migrations/007_add_task_types_and_learning.sql`
- [ ] העתקתי את **כל** התוכן (Ctrl+A, Ctrl+C)
- [ ] הדבקתי ב-SQL Editor (Ctrl+V)
- [ ] לחצתי "Run" (או Ctrl+Enter)
- [ ] הופיעה הודעה: "Success. No rows returned" ✅

### בדיקה
- [ ] בדקתי שהשדה `task_type` קיים (ראה למטה)
- [ ] בדקתי שהטבלה `task_type_stats` נוצרה
- [ ] בדקתי שהטבלה `task_completion_history` נוצרה
- [ ] בדקתי שהטריגר `update_task_type_stats_trigger` קיים

#### קוד לבדיקה:
```sql
-- בדיקה 1: שדה task_type
SELECT column_name FROM information_schema.columns 
WHERE table_name = 'tasks' AND column_name = 'task_type';
-- תוצאה צפויה: שורה אחת

-- בדיקה 2: טבלת סטטיסטיקות
SELECT COUNT(*) FROM information_schema.tables 
WHERE table_name = 'task_type_stats';
-- תוצאה צפויה: 1

-- בדיקה 3: טבלת היסטוריה
SELECT COUNT(*) FROM information_schema.tables 
WHERE table_name = 'task_completion_history';
-- תוצאה צפויה: 1

-- בדיקה 4: טריגר
SELECT trigger_name FROM information_schema.triggers 
WHERE trigger_name = 'update_task_type_stats_trigger';
-- תוצאה צפויה: שורה אחת
```

### בדיקת הטריגר (אופציונלי)
```sql

SELECT COUNT(*) FROM task_completion_history WHERE user_id = 'YOUR_USER_ID';
-- אמור להיות מספר המשימות שסיימת

-- בדוק שיש סטטיסטיקות
```

## 🐛 פתרון בעיות

### ❌ לא רואה "סוג משימה" בטופס

**בדוק:**
- [ ] הרצתי את ה-migration?
- [ ] רעננתי את הדפדפן (Ctrl+F5)?
- [ ] יש שגיאות ב-Console (F12)?

**פתרון:**
1. הרץ את ה-migration שוב
2. רענן (Ctrl+F5)
3. נקה Cache
4. בדוק את קובץ `.env` שכל ההגדרות נכונות

### ❌ שגיאה: "column task_type does not exist"

**פתרון:**
- ה-migration לא רץ. הרץ אותו שוב.

### ❌ החיזויים לא עובדים

**בדוק:**
- [ ] השלמתי לפחות 3 משימות מאותו סוג?
- [ ] השתמשתי בטיימר?
- [ ] סימנתי כהושלם?

**פתרון:**
- צריך לפחות 3 משימות כדי שהחיזוי יעבוד
- ודא שהטריגר רץ (ראה בדיקות למעלה)

### ❌ עמוד התובנות ריק

**בדוק:**
- [ ] השלמתי לפחות 10 משימות?
- [ ] בחרתי סוגי משימות (לא רק "אחר")?

**פתרון:**
- תובנות מפורטות דורשות לפחות 10 משימות
- צריך מגוון של סוגים

### ❌ שגיאות אחרות

**בדוק:**
- [ ] Console Browser (F12)
- [ ] Logs ב-Supabase Dashboard
- [ ] טבלת RLS Policies

**פתרון:**
1. צלם screenshot של השגיאה
2. העתק את הודעת השגיאה
3. קרא [פתרון בעיות](WHATS_NEW.md#-פתרון-בעיות-נפוצות)
4. פתח issue ב-GitHub
