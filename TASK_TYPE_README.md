# 🎯 מערכת סיווג משימות ולמידה אישית

## 📌 מבוא

האפליקציה שודרגה עם מערכת למידה מתקדמת שעוזרת לך להכיר את עצמך טוב יותר ולשפר את ניהול הזמן שלך.

**במקום לנחש כמה זמן משימה תיקח, המערכת תדע בדיוק!**

## 🚀 התחלה מהירה

### 1️⃣ הרץ את ה-Migration
```bash
# ב-Supabase SQL Editor, הרץ:
supabase/migrations/007_add_task_types_and_learning.sql
```
📖 [הוראות מפורטות](MIGRATION_INSTRUCTIONS.md)

### 2️⃣ רענן את האפליקציה
```bash
# במסוף הפרויקט:
npm run dev
```

### 3️⃣ התחל להשתמש!
1. צור משימה חדשה
2. בחר **סוג משימה** (🎙️ תמלול, ✍️ הגהה, וכו')
3. השתמש ב**טיימר** לעקוב אחר הזמן
4. סיים את המשימה
5. ראה את ה**תובנות** בעמוד המיוחד!

## 🎓 דוגמאות שימוש

### יצירת משימה עם חיזוי

```javascript
import { useTasks } from './hooks/useTasks';
import { predictTaskDuration } from './utils/taskTypeLearning';

// קבלת חיזוי
const prediction = await predictTaskDuration(
  userId, 
  'transcription'
);

// יצירת משימה
await addTask({
  title: 'תמלול הרצאה',
  taskType: 'transcription',
  estimatedDuration: prediction.predictedTime
});
```

### הצגת תובנות

```javascript
import { getTaskTypeInsights } from './utils/taskTypeLearning';

const insights = await getTaskTypeInsights(userId, 'transcription');

insights.insights.forEach(insight => {
  console.log(`${insight.icon} ${insight.title}: ${insight.message}`);
});
```
