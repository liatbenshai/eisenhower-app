# 🚀 התחל כאן - מערכת סיווג משימות ולמידה

## 👋 שלום!

האפליקציה שלך שודרגה עם מערכת למידה חכמה שתעזור לך לנהל את הזמן טוב יותר!

## ⚡ 3 צעדים פשוטים להתחלה

### 1️⃣ הרץ את ה-Migration (חובה!)

**מה זה?** קובץ SQL שמוסיף טבלאות חדשות למסד הנתונים.

**איך?**
1. פתח https://app.supabase.com
2. בחר את הפרויקט שלך
3. לחץ **SQL Editor** (בתפריט הצד)
4. לחץ **+ New Query**
5. פתח את הקובץ: `supabase/migrations/007_add_task_types_and_learning.sql`
6. **העתק הכל** (Ctrl+A, Ctrl+C)
7. **הדבק** ב-SQL Editor (Ctrl+V)
8. לחץ **Run** (או Ctrl+Enter)
9. ✅ אמור להופיע: "Success. No rows returned"

**עזרה?** → [הוראות מפורטות](MIGRATION_INSTRUCTIONS.md)

### 2️⃣ רענן את האפליקציה

```bash
# אם זה רץ מקומית:
npm run dev

# או פשוט רענן את הדפדפן
F5
```

### 3️⃣ התחל להשתמש!

1. **צור משימה חדשה** 📝  
   לחץ על "הוסף משימה"

2. **בחר סוג משימה** 🏷️  
   תראה 8 סוגים: תמלול, הגהה, תקשורת וכו'  
   המערכת גם תזהה אוטומטית!

3. **השתמש בחיזוי** 🤖  
   המערכת תציע לך זמן משוער מדויק

4. **עבוד עם הטיימר** ⏱️  
   לחץ "התחל עבודה" והטיימר ימדוד בדיוק

5. **סיים את המשימה** ✅  
   סמן כהושלם - המערכת תלמד מזה!

6. **ראה תובנות** 📊  
   תפריט ≡ → "תובנות למידה"

## 💡 מה המערכת עושה?

### בפשטות:

1. **אתה יוצר משימה** → המערכת מזהה מה הסוג
2. **אתה עובד** → הטיימר שומר את הזמן בדקות
3. **אתה מסיים** → המערכת לומדת כמה זה לקח לך
4. **בפעם הבאה** → המערכת כבר יודעת לחזות!

### לדוגמה:

```
📝 משימה 1: "תמלול פגישה"
   אתה עובד: 45 דקות
   המערכת: 💾 "זכרתי!"

📝 משימה 2: "תמלול אחר"
   המערכת: 🤖 "אצלך זה לוקח בערך 45 דקות"
   אתה עובד: 48 דקות
   המערכת: ✅ "דיוק 94%!"

📝 משימה 5: "תמלול נוסף"
   המערכת: 🤖 "47 דקות (ממוצע של 5 משימות)"
   המערכת: 🟢 "רמת ביטחון: גבוהה"
   המערכת: 📊 "דיוק שלך: 91%"

📊 אחרי שבועיים:
   "אתה הכי טוב ביום שלישי בשעה 10:00"
   "משימות תמלול אצלך: 47±3 דקות"
   "המשך כך - אתה מאוד עקבי! 🎉"
```

## ❓ שאלות נפוצות

### "צריך להריץ את ה-migration?"
**כן!** בלי זה המערכת לא תעבוד. זה לוקח 30 שניות.

### "זה בטוח?"
**כן!** כל הנתונים שלך בלבד. אף משתמש אחר לא רואה אותם.

### "מתי אראה תוצאות?"
אחרי 3-5 משימות מאותו סוג תתחיל לראות חיזויים טובים.  
אחרי 10-15 משימות תקבל תובנות מפורטות.

### "אני צריך לבחור סוג משימה?"
**כן**, אבל המערכת עוזרת! היא מזהה אוטומטית לפי מילות מפתח.

### "מה אם אני לא רוצה להשתמש בזה?"
בסדר גמור! פשוט אל תבחר סוג משימה. המערכת תמשיך לעבוד רגיל.

### "איך אני מבטל את זה?"
יש הוראות למחיקה ב-[MIGRATION_INSTRUCTIONS.md](MIGRATION_INSTRUCTIONS.md)  
(אבל אנחנו מקווים שלא תצטרך!)
